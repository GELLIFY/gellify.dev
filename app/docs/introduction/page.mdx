# Introduction

Welcome to the documentation for our Next.js Enterprise Template! This template is a production-ready foundation for building scalable web applications using modern technologies.

## What's Included

Our template combines best-in-class technologies and practices:

- **Next.js 14+** - React framework with App Router and Server Components
- **TypeScript** - For type safety and better developer experience
- **Shadcn/UI** - High-quality, customizable UI components
- **NeonDB** - Serverless Postgres database
- **Drizzle ORM** - TypeScript ORM for SQL databases
- **Clerk** - Authentication and user management
- **Tailwind CSS** - Utility-first CSS framework
